# ITAHome 后端开发规划文档

## 项目概述

ITAHome 是一个基于 Spring Boot 的后端管理系统，采用分阶段迭代开发的方式进行功能实现。

## 技术栈

- **后端框架**: Spring Boot 3.x
- **数据库**: MySQL 8.0 + MyBatis Plus  
- **认证**: JWT
- **缓存**: Caffeine (本地缓存)
- **定时任务**: Spring Scheduler
- **异步处理**: CompletableFuture + 线程池
- **外部API**: RestTemplate
- **构建工具**: Maven
- **Java版本**: JDK 17+
