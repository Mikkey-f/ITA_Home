# ITAHome 后端开发规划文档

## 项目概述

ITAHome 是一个基于 Spring Boot 的后端管理系统，采用分阶段迭代开发的方式进行功能实现。

## 开发计划安排

### 📅 Epoch 1 - 用户认证基础 ✅ **已完成**

**功能范围：**
- 用户注册/登录功能
- JWT 权限鉴权系统  
- 邮箱验证码发放机制

**当前状态：** 已完成开发和测试，功能正常运行

## 技术栈

- **后端框架**: Spring Boot 3.x
- **数据库**: MySQL 8.0 + MyBatis Plus  
- **认证**: JWT
- **构建工具**: Maven
- **Java版本**: JDK 17+
